# Materials

Простой репозиторий материалов.

**Makefile**:

- `build`: цель по умолчанию, компилирует все **.typ* в *.pdf*, *.html* и *.md*.
- `pdfs`: генерирует только PDF-файлы.
- `htmls`: генерирует только HTML-файлы.
- `md`: конвертирует HTML в Markdown с помощью **pandoc**.
- `clean`: удаляет сгенерированные файлы (*.pdf*, *.html*, *.md*).

Структура проекта:

- **.typ* — исходные файлы **Typst** .
- *lib/* — вспомогательные компоненты.
- *img/* — изображения, используемые в материалах.
- **Makefile** — сценарии сборки/экспорта.

Использование:

- `make` или `make build` — собрать все артефакты
- `make clean` — удалить сгенерированные файлы
